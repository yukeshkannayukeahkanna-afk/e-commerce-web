# Organic Skincare E-Commerce Website

## Overview

This project is a modern **Organic Skincare E-Commerce Website** developed to showcase and sell natural skincare products online.
The website focuses on a clean user interface, smooth navigation, and an engaging shopping experience.

## Features

* Responsive modern UI design
* Product listing with images and details
* Add to Cart functionality
* Interactive user interface
* Smooth animations and hover effects
* User-friendly navigation

## Technologies Used

* HTML
* CSS
* JavaScript

1. Download or clone the repository
2. Open the project folder
3. Run **index.html** in a web browser
